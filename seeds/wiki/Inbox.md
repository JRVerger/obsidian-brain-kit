# Inbox

Captura rapida — procesa esto regularmente (cada weekly review) y mueve a su sitio.

## Quick Capture

> QuickAdd (Mod+J) → "Inbox Capture" anade aqui automaticamente con timestamp.

## Para Procesar

- [ ] (vacio — las capturas iran apareciendo aqui)

## Como procesar

1. Cada viernes (weekly review): abre Inbox
2. Cada item -> una de 3 acciones:
   - **Archivar**: ya no aplica, marcar done y listo
   - **Mover**: convertir en note formal en `concepts/`, `learning/`, `questions/`, etc.
   - **Ejecutar**: crear task en proyecto o daily note, marcar done aqui
3. Meta: Inbox vacio cada fin de semana
